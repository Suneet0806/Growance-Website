# Grovance – Campus Marketplace

**Grovance** is a campus-verified marketplace built for college students in India to safely buy and sell pre-owned textbooks, electronics, furniture, and other items related to student life. It aims to create a secure and community-focused trading platform, helping students save money, avoid scams, and promote sustainability.

## 🚀 Features

- 📚 Buy and sell pre-owned textbooks and academic materials  
- 📱 List electronics, project gear, furniture, and accessories  
- 🛡️ Campus-verified user base for safer transactions  
- 💸 Helps students save money and avoid scams  
- 🌱 Encourages reuse and sustainability in student communities

## 💡 Why Grovance?

College students often need reliable ways to trade used items within their community. **Grovance** provides:

- A trusted marketplace tailored for student needs  
- A simple landing page to introduce the platform concept  
- A prototype to showcase core ideas and design

## 🧠 Tech Stack

- ✨ HTML/CSS for the landing page and prototype  
- 🔧 Lightweight and easy to extend for full stack development
